## STUDENT GRADE MANAGEMENT SYSTEM
This lab is a console application in Java for managing students, subjects and grades
The features it implements are:
- Student registration
- Viewing all students
- Recording grades for bith core and elective subjects
- Viewing grade reports
- Menu navigation
- 
The console application demonstrates these OOP principles: 
- Abstract classes
- Inheritance
- Arrays for data storage
- Method overriding
- Composition
- Interface
- Polymorphism

The lab consists of nine (9) classes as shown in the src file

## Setup Instructions
-  Java JDK and any IDE is required to run this lab
- Clone repository
- Run Main.java
